# 🔍 Basic Network Scanner

A Python-based network scanner that detects active devices in a local network using ARP requests.

---

## 🚀 Features
- Scan a given IP range
- Detect active devices in the network
- Display IP and MAC addresses
- Fast and simple CLI tool

---

## 🛠️ Technologies Used
- Python
- Scapy

---

## 📦 Installation

1. Clone the repository:
git clone https://github.com/abi0216/Basic-Network-Scanner.git

2. Navigate to the project folder:
cd Basic-Network-Scanner

3. Install dependencies:
pip install -r requirements.txt

---

## ▶️ Usage

Run the scanner using:

python scanner.py -t 192.168.1.1/24

---

## 📸 Output Example

Example result:

Developed a Python-based network scanner using ARP protocol to identify active devices in a local network. Implemented IP range scanning and MAC address detection using Scapy.
